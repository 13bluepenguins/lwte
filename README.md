# LWTE

Lightweight Terminal Editor written in C.

## Features

- Open existing files
- Create new files
- Save files
- Multi-line editing
- Arrow key navigation
- Vertical and horizontal crolling
- Terminal-native copy/paste
- Search

# Instalation Guide for Linux

run git clone https://github.com/13bluepenguins/lwte.git 
then 
cd lwte

## Build

make

## Install

sudo make install

## Run

lwte filename.txt

or:

lwte

to create/open a file.
