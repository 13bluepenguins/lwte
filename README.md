# LWTE

Lightweight Terminal Editor written in C.

## Features

- Open existing files
- Create new files
- Save files
- Multi-line editing
- Arrow key navigation
- Vertical and horizontal crolling
- Terminal-native copy/paste
- Search

# Instalation Guide for Linux

Run: 

```bash
git clone https://github.com/13bluepenguins/lwte.git 
```

then 

```bash
cd lwte
```

## Build

```bash
make
```

## Install

```bash
sudo make install
```

## Run

```bash
lwte filename.txt
```

or:

```bash
lwte
```

to create/open a file.
